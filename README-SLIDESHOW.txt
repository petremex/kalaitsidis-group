# Οδηγίες Ενημέρωσης Slideshow

## Πώς να προσθέσετε νέες φωτογραφίες:

1. Προσθέστε τις φωτογραφίες σας στον φάκελο: `photos\slideshow\`

2. Κάντε διπλό κλικ στο αρχείο: **update-slideshow.bat**

3. Ανανεώστε την σελίδα στον browser (F5)

## Υποστηριζόμενοι τύποι αρχείων:
- JPG / JPEG
- PNG
- GIF
- WEBP

# Slideshow Update Instructions

## How to add new photos:

1. Add your photos to the folder: `photos\slideshow\`

2. Double-click the file: **update-slideshow.bat**

3. Refresh the page in your browser (F5)

## Supported file types:
- JPG / JPEG
- PNG
- GIF
- WEBP
